🚀 HOW TO USE (3 SIMPLE STEPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. OPEN
   → Double-click "index.html"
   → Works in any browser (Chrome, Firefox, Safari, Edge)

2. SELECT
   → Choose your key, scale, tempo
   → Pick a chord progression (100+ presets!)
   → Select song structure
   → Enable instrument tracks

3. GENERATE
   → Click "GENERATE MIDI"
   → File downloads automatically
   → Import into your DAW and make music!

🎯 EXAMPLE: CREATE A POP SONG
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Section 1 (Basic Settings):
   Key: C
   Scale: Major
   Tempo: 120 BPM

2. Section 2 (Chord Progression):
   Click "POPULAR" tab
   Select "I-V-vi-IV" (most popular progression ever!)

3. Section 3 (Song Structure):
   ✓ Intro
   ✓ Verse (2 verses)
   ✓ Chorus (3 choruses)
   ✓ Outro

4. Section 4 (Instrumentation):
   ✓ Drums
   ✓ Bass
   ✓ Chords
   ✓ Lead

5. Click "GENERATE MIDI"

6. Done! File downloads to your Downloads folder
